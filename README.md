# AdFlow Pro - React + Supabase Project

## Project Description
This is a simple Ad Management System built using React and Supabase.  
Users can add ads and view all posted ads in real time.

---

## Features
- Add Ads (Title & Description)
- View Ads List
- Supabase Database Integration
- Responsive Dashboard UI

---

## Tech Stack
- React (Vite)
- Supabase
- JavaScript
- Tailwind CSS
